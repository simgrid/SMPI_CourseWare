SMPI simulates the execution of MPI applications by relying on the fast and accurate simulation core provided by SimGrid. Consequently, the SMPI user (you!) must describe as simulated platforms (i.e., a set of hosts and network links, with some performance characteristics). The SimGrid documentation provides ample information on platform descriptions, which are done in XML. Below we simply show a series of examples, which should be sufficient for our purpose. Note that platform files are typically provided in each pedagogic module, but you may have to modify them.

By default, routes are symmetrical. See more information on the SimGrid Web site.
